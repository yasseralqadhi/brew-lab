# ☕ Brew Lab — مختبر القهوة المختصة

تطبيق ويب تفاعلي يولّد وصفات قهوة مختصة مخصصة بناءً على معلومات المحصول والمعدات والماء والتفضيلات الذوقية.

## المميزات

- **محرك طحن بنظام الميكرون** — يحسب الميكرون المستهدف لكل طريقة تحضير، ثم يحوّله إلى إعداد كل مطحنة باستخدام معاملات معايرة موثّقة (ZP6: 22µm/كليك، J-Ultra: 8µm/كليك، Ode Gen 2: ~25µm/خطوة، وغيرها)
- **رفع صور** لكيس البن وعبوة الماء لتسهيل التعبئة (الاستخراج التلقائي للمعلومات قيد التطوير)
- **ملفات محفوظة** — احفظ ملف بن وملف ماء واستدعِهما لاحقاً بدل إعادة الإدخال كل مرة
- **تحليل الماء** مع تقييم لوني لكل معدن
- **توقّع النكهة** بناءً على المنشأ × المعالجة × عمر التحميص
- **جدول صبات مفصّل** + مؤقت تحضير مدمج
- دعم كامل للعربية (RTL) وتصميم متجاوب

## ملاحظة منهجية

تحويل الميكرون إلى إعداد مطحنة هو **نقطة انطلاق** وليس رقماً نهائياً — نظام البر غير خطّي تماماً، ودبس البن المحدد ودرجة تحميصه يؤثران على النتيجة. التطبيق يعرض نقطة الانطلاق ونطاق المعايرة، ويترك الضبط النهائي للتذوق.

## التشغيل

افتح `index.html` في أي متصفح حديث — لا يحتاج بناء أو تثبيت.

## النشر على GitHub Pages

Settings → Pages → Source: `Deploy from a branch` → `main` / `root`
